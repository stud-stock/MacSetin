1. Собрать топологию по рисунку Lab_08_1.png
Использовать маршрутизаторы:
AR1-AR3 - AR1220
R1-R4   - Router (без номера модели)

2. Настроить ВMА-сеть:
- назначить IP адреса, указанные на рисунке (Использовать сеть 192.168.N.0/24)
- настроить OSPF протокол на AR1 AR2 AR3 R1, МВА сеть - магистральная зона, 
  при этом Router ID вручную не задавать, маршрутизаторы выберут их сами.
- результаты выборов DR/BDR будут зависеть от порядка настройки маршрутизаторов.

3. вопросы:
- какие Router ID у маршрутизаторов магистральной зоны?
- почему они такие?
- какие типы соседства у DR-BDR, DR-Other, BDR-Other, Other-Other пар?

4. выключить хаб и включить снова.
- дождаться окончания работы OSPF (установления full-соседства)
- какие маршрутизаторы стали DR/BDR?
- почему?

5. На R1 на интерфейсе, направленном в ВMА сеть задать приоритет:
[Huawei-GigabitEthernet0/0/0]ospf dr-priority 100

6. Повторить пункт 4.

7. На AR1 задать Router ID вручную и перезапустить OSPF процесс:
[Huawei]router id 200+N.200+N.200+N.200+N
<Huawei>reset ospf 1 process 

8. Выполнить на R1 просмотр OSPF соседей
- каковы Router ID соседей?
- кто является DR и BDR?
- почему? 

9. Повторить пункт 4.

10. Настроить Р2Р сеть между R1 и R4
- назначить IP адреса
- настроить OSPF, зона 1

11. Убедится, что на на маршрутизаторах AR1 AR2 AR3 R4 в таблицах маршрцтизации 
появилась новая сеть.

12. На R1 и R4:
- каков тип соседства в Р2Р сети?
- каков результат выбора DR/BDR?

13. Настроить NBMA сеть (Frame Relay):
R1 (по умолчанию на FR линках будет работать INARP):
[Huawei-Serial0/0/0]ip address 20.0.N.1 255.255.255.0
[Huawei-Serial0/0/0]link-protocol fr
[Huawei-Serial0/0/0]fr dlci 100
[Huawei-Serial0/0/0]fr dlci 101

R2:
[Huawei-Serial0/0/0]ip address 20.0.N.2 255.255.255.0
[Huawei-Serial0/0/0]link-protocol fr
[Huawei-Serial0/0/0]fr dlci 100
[Huawei-Serial0/0/0]fr dlci 102

R3:
[Huawei-Serial0/0/0]ip address 20.0.N.3 255.255.255.0
[Huawei-Serial0/0/0]link-protocol fr
[Huawei-Serial0/0/0]fr dlci 101
[Huawei-Serial0/0/0]fr dlci 102

FR-коммутатор настраивается в соответствии с рисунком Lab_08_1_1.png

14. Включить захват пакетов на одном из Serial-интерфейсов в FR-сети
- проверить связность FR сети с маршрутизатора, на интерфейсе запущен Wireshark
- в перехваченном FR трафике пронаблюдать процесс разрешения IP адреса c помощью INARP
- выполнить просмотр состояния FR сети:
display fr inarp-info
display fr map-info 
display fr pvc-info

15. Настроить на R1 R2 и R3 OSPF, зона 2
- соседство установлено?

16. Настроить соседство в NBMA сети:
На R1:
[Huawei-ospf-1]peer 20.0.N.2
[Huawei-ospf-1]peer 20.0.N.3

На остальных маршрутизаторах NBMA сети - аналогично.

17. Выполнить просмотр соседей на R3.
- соседство установлено?
- какого типа соседство?
- есть ли DR/BDR?
- почему так произошло?
