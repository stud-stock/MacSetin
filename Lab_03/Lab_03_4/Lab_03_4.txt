1. Разделить сеть 192.168.N.0/24 на части, указанного на рисунке Lab_03_4.png размера (использовать маски различной длины).
- serial линкам выделять блоки адресов с маской /30

2. Собрать топологию, по рисунку Lab_04_4.png
- Для сборки топологии следует брать маршрутизатор "Router"(без буквенно-цифрового индекса) 
- только у него достаточное количество (serial) адаптеров

3. Периферийные сети для упрощения топологии эмулировать петлевыми (loopback) адаптерами. 
- Петлевые адаптеры создаются в момент начала их настройки
- в качестве адреса интерфейса следует взять первый адрес соответствующего диапазона

[R1]interface LoopBack 0
[R1-LoopBack0]ip address ...

4. Настроить статическую маршрутизацию для получения полной связности сети.

5. Сделать скриншот таблицы маршрутизации на верхнем маршрутизаторе.
